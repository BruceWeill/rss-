# RSS新闻获取与翻译工具

这个项目可以自动从RSS源获取新闻，并使用大语言模型将新闻内容翻译成中文。

## 功能特点

- 从多个RSS源获取新闻资讯
- 自动处理SSL连接问题，提高获取成功率
- 使用大语言模型进行高质量翻译
- 支持GMT时间到北京时间的自动转换
- 批量处理提高翻译效率

## 文件说明

- `1.py` - RSS新闻获取脚本，从指定RSS源获取新闻数据
- `2.py` - 新闻翻译脚本，将获取的英文新闻翻译成中文
- `news/` - 存放从RSS获取的原始新闻数据
- `translated_news/` - 存放翻译后的新闻数据
